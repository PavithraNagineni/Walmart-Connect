# 🛒 WalmartConnect - Task Dashboard

A modern **React-based task management dashboard** designed for retail store operations. This application helps store associates efficiently manage daily tasks like restocking, cleaning, inventory updates, and customer assistance with real-time tracking and team coordination.

---

## 🚀 Features

* 📋 Task Management (Create, Start, Complete, Reopen)
* 👥 Team Assignment & Status Tracking
* 🔍 Search Tasks Functionality
* 📊 Real-time Progress Overview
* 🔔 Notifications Panel
* 📝 Recent Activity Tracking
* 📱 Responsive UI Design
* 🎯 Category-based Tasks (Restocking, Cleaning, Inventory, etc.)

---

* **Frontend:** React.js

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/WalmartConnect-Task-Dashboard.git
```

2. Navigate to project folder

```bash
cd WalmartConnect-Task-Dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

5. Open in browser

```
http://localhost:3000
```
